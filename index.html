More changes coming in

Project's going strong, we some new features already implemented. Team members are getting more varied - with gear and skills. Inventory management works, you can swap items around and force a reload on a weapon. Skills and other items still need to be implemented, but we're getting there. And fast.

You can check out the latest at http://ethnar.com/Zombielab/

Latest version available online

After some more work, we have walking zombies, Game Overs and option to actually beat the levels. Still miles to go and current state doesn't really resemble the final gameplay, but you can check it out online nonetheless. Latest build is available at: http://ethnar.com/Zombielab/#/game

Welcome to ZombieLab

The website is about a new game in works called (temporarily?) ZombieLab. The game is a dungeon crawler with some RPG elements, primarily targeted for mobile devices.
