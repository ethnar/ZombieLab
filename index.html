More updates coming in, graphics & more

Rooms, walls and doors now have their graphics. There's still lots of work needed to put stuff inside the rooms so that each type of the room is recognizable, but we have a good start now. New stuff, like access cards, fire, explosives barrels and flashlights were added to the game. Same as with graphics - that's far from complete list of features, but you will have a better feel of how this game plays out.

Latest version is available at: http://ethnar.com/Zombielab/stable/

Milestone reached

All the core components for the game are now in. We have items, different room types, zombie spawners and even a tutorial. There's still lots of work to be done (game balance, random events, items, performance improvements etc.), but it's time to reach out and get some help to replace this excuse of graphics with [hopefully] something very plausible.

Also, from now on I'll be keeping a more stable version in a separate location, with most recent dev version available at the usual location.

Latest [stable] version is available at: http://ethnar.com/Zombielab/stable/

Going strong

Been a little while since I've updated this log, but it was still really busy in here. The latest version has lots of improvements, starting with items that actually work, skills that actually affect your performance, information windows (hold tap/mouse to show character/item info), animations, performance improvements and lots more.

The latest is available at: http://ethnar.com/Zombielab/

More changes coming in

Project's going strong, we some new features already implemented. Team members are getting more varied - with gear and skills. Inventory management works, you can swap items around and force a reload on a weapon. Skills and other items still need to be implemented, but we're getting there. And fast.

You can check out the latest at http://ethnar.com/Zombielab/

Latest version available online

After some more work, we have walking zombies, Game Overs and option to actually beat the levels. Still miles to go and current state doesn't really resemble the final gameplay, but you can check it out online nonetheless. Latest build is available at: http://ethnar.com/Zombielab/#/game

Welcome to ZombieLab

The website is about a new game in works called (temporarily?) ZombieLab. The game is a dungeon crawler with some RPG elements, primarily targeted for mobile devices.
